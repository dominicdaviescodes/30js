Day 1 Split Method

Dom Davies

split() method splits a string into an array of substrings.

Takes 2 optional arguements. (separator, limit). separator - determines the character to use to split the string. If blank returns our string as an array. Can be a letter or the space (' ') which returns each word as an array. .

Limit - Optional. An integer that specifies the number of splits, items after the split limit will not be included in the array. Example, if you split a string (" ") by it's spaces and put in a limit of 3, the array returns the 1st 3 words only
